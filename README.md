# Byggkostnadskalkylatorn

En plugin för att beräkna kostnader baserat på olika parametrar inom byggbranschen.

## Installation

1. Ladda ner pluginen till din WordPress-webbplats.
2. Aktivera pluginen från WordPress-adminpanelen.

## Användning

Denna plugin lägger till en kortkod `[calculation_form]` som du kan använda för att infoga beräkningsformuläret på en sida eller inlägg på din WordPress-webbplats. Besökare kan sedan välja en kategori, ange antalet kvadratmeter och välja typ av material för att beräkna kostnaden för sitt byggprojekt.
